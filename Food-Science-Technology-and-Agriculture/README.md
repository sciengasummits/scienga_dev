# Food Science Technology and Agriculture

A comprehensive web application designed for the Food Science Technology and Agriculture summit, facilitating event management, attendee registration, and information dissemination regarding advancements in food science and agricultural technologies.

## Features

- **Event Overview**: detailed information about the summit, including theme, mission, and vision.
- **Schedule & Program**: Interactive schedule of events, sessions, and workshops.
- **Registration System**: User-friendly registration form for attendees.
- **Venue & Travel**: Information about the venue, location maps, and travel guidelines.
- **Sponsorship**: Details for sponsors and partners.
- **Responsive Design**: Optimized for viewing on desktops, tablets, and mobile devices.

## Tech Stack

- **Frontend Framework**: [React](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: CSP-compatible CSS with CSS Variables for theming.
- **Routing**: [React Router](https://reactrouter.com/)

## Getting Started

### Prerequisites

- Node.js (v16 or higher recommended)
- npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/sciengasummits/scienga_dev.git
    ```

2.  Navigate to the project directory:
    ```bash
    cd Food-Science-Technology-and-Agriculture
    ```

3.  Install dependencies:
    ```bash
    npm install
    ```

### Running the Application

To start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

### Building for Production

To create a production build:

```bash
npm run build
```

The built files will be in the `dist` directory.

## Project Structure

- `src/components`: Reusable UI components (forms, layout, sections).
- `src/pages`: Page components corresponding to routes.
- `src/assets`: Static assets like images and styles.
- `src/App.jsx`: Main application component and routing setup.
- `src/main.jsx`: Entry point.
